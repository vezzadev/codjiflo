# Milestone Archive

Historical delivery plans for the seven shipping milestones. Each file describes the scope, scaffolding, and acceptance criteria for what shipped — preserved for narrative context. The canonical "what the system does today" lives in `openspec/specs/` (see `openspec/changes/` for in-flight work).

- [M1: SPA + GitHub Data](milestone-1-spa-github-data.md) — established app shell, GitHub auth, basic diff
- [M2: Comments Engine](milestone-2-minimal-comments.md) — inline commenting + threads
- [M3: Advanced Diff Engine](milestone-3-advanced-diff.md) — side-by-side, word-level diffs, Web Worker
- [M4: Iteration Management](milestone-4-iteration-management.md) — GitHub Action + SQLite artifact (stateful mode)
- [M4.1: Unauthenticated Experience](milestone-4.1-unauthenticated-experience.md) — public PR review without login
- [M4.2: Stateless Iteration Management](milestone-4.2-stateless-iteration-management.md) — Timeline API + Web Worker (stateless mode)
- [M5: Full Comments & Canvas Layouts](milestone-5-full-comments.md) — floating bubble layout engine
- [M6: Real-Time & Polish](milestone-6-remaining-features.md) — delta polling, ETag/Last-Modified
- [M7: Extension Bridge](milestone-7-browser-extension.md) — content-script injection into GitHub
